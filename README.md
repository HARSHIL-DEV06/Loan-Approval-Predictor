# Loan Approval Prediction System

## 📌 Overview

This project is a machine learning-based application that predicts whether a loan application will be approved or not based on applicant details.

It was developed as a mini project to understand and implement the complete machine learning workflow — from data analysis to model deployment.

---

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature analysis to understand key factors affecting loan approval
* Model building and comparison
* Deployment using Streamlit
* Hosted on Render for public access

---

## 🧠 Machine Learning Workflow

The project follows a structured ML pipeline:

1. **Data Analysis**

   * Explored dataset to understand distributions and relationships
   * Identified important features influencing loan approval

2. **Feature Engineering**

   * Handled categorical and numerical variables
   * Converted relevant columns for better model performance

3. **Model Training & Comparison**

   * Compared models based on accuracy score:

     * Logistic Regression
     * Decision Tree Classifier
     * Random Forest Classifier

4. **Model Selection**

   * Random Forest performed best among all models
   * Selected as the final model

5. **Model Evaluation**

   * Evaluated models using standard metrics (accuracy, etc.)
